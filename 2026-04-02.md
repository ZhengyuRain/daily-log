# 2026-04-02 | Fama-French 三因子模型复现

## 今日主题

复现 Fama & French (1993) 三因子模型，并用 Python 对单支股票做回归分析。

> **Paper**: Fama, E. F., & French, K. R. (1993). Common risk factors in the returns on stocks and bonds. *Journal of Financial Economics*, 33(1), 3–56.

---

## 模型简介

FF3 在 CAPM 的基础上加入了两个额外风险因子：

```
R_i - R_f = α + β₁(R_m - R_f) + β₂·SMB + β₃·HML + ε
```

| 因子 | 含义 |
|------|------|
| `Rm - Rf` | 市场超额收益（Market Premium） |
| `SMB` | Small Minus Big，规模溢价 |
| `HML` | High Minus Low，价值溢价 |

---

## Python 实现

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm
import pandas_datareader.data as web
from datetime import datetime

# ── 1. 从 Ken French 数据库获取因子数据 ──────────────────────────
start, end = datetime(2015, 1, 1), datetime(2024, 12, 31)

ff3 = web.DataReader("F-F_Research_Data_Factors", "famafrench", start, end)[0]
ff3.index = pd.to_datetime(ff3.index.to_timestamp())
ff3 = ff3 / 100  # 转为小数

# ── 2. 获取目标股票月收益（示例：AAPL） ─────────────────────────
import yfinance as yf

prices = yf.download("AAPL", start=start, end=end, interval="1mo", auto_adjust=True)["Close"]
returns = prices.resample("ME").last().pct_change().dropna()
returns.index = returns.index.to_period("M").to_timestamp()

# ── 3. 合并数据 ───────────────────────────────────────────────────
df = pd.concat([returns.rename("R_stock"), ff3], axis=1).dropna()
df["excess_return"] = df["R_stock"] - df["RF"]

# ── 4. OLS 回归 ───────────────────────────────────────────────────
X = sm.add_constant(df[["Mkt-RF", "SMB", "HML"]])
y = df["excess_return"]

model = sm.OLS(y, X).fit()
print(model.summary())

# ── 5. 解读结果 ───────────────────────────────────────────────────
alpha = model.params["const"]
beta_mkt = model.params["Mkt-RF"]
beta_smb = model.params["SMB"]
beta_hml = model.params["HML"]

print(f"\nAlpha (年化): {alpha * 12:.4f}")
print(f"Beta 市场:    {beta_mkt:.4f}")
print(f"Beta SMB:     {beta_smb:.4f}  ({'偏小盘' if beta_smb > 0 else '偏大盘'})")
print(f"Beta HML:     {beta_hml:.4f}  ({'偏价值' if beta_hml > 0 else '偏成长'})")
```

